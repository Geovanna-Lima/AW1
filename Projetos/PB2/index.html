Perdidos no Saber

O Projeto Perdidos no Saber foi desenvolvido para atender à proposta de trabalho das disciplinas técnicas do 1° ano do curso de Informática Integrado no IFSP.

O jogo Perdidos no Saber foi criado para auxiliar crianças de 10 a 12 anos. Conta com sete fases que ajudam a criança a aprender de forma lúdica, o projeto abordava as matérias básicas do currículo escolar.

Informações Técnicas:

O jogo foi desenvolvido na linguagem GML pelo software de desenvolvimento Game Maker Studio.

HealthCare

O Projeto HealthCare foi desenvolvido para atender à proposta de trabalho com banco de questões pela disciplina de Projetos do 2° ano do curso de Informática Integrado no IFSP.

A plataforma conta com mais de 30 questões divididas igualmente entre as doenças: infecciosas, respiratórias e transtornos mentais, além de gráficos de desempenho para que o usuário visualize o seu aproveitamento.

Informações Técnicas:

O site foi desenvolvido com o uso das linguagens HTML, CSS, JavaScript, PHP, Node.js e MySQL. Além dos frameworks Bootstrap e Chart.js.

Conexão Cultura

O Projeto Conexão Cultura está sendo desenvolvido para atender à proposta de trabalho com C# pela disciplina de Projetos do 3° ano do curso de Informática Integrado no IFSP.

A plataforma irá conectar a população com o estado de São Paulo recomendando pontos turísticos, restaurantes, parques, eventos e etc.

Informações Técnicas:

O site está sendo desenvolvido com o uso das linguagens HTML, CSS, JavaScript, PHP, C# e MySQL. Além do framework Bootstrap.
